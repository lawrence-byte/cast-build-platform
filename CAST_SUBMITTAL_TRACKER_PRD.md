# CAST Submittal Tracker PRD

## Product

**Project:** CAST Project Controls Platform  
**Initial project:** Broderick  
**Module:** Submittal Tracking

## Objective

Build a professional construction submittal tracking MVP for creation, packages, routing, sequential/parallel review, ball-in-court responsibility, due dates, responses, official response, revisions, closeout readiness, and exportable logs.

## MVP scope

- Submittal dashboard and dense log table.
- Create draft, required placeholder, and submitted-for-review records.
- Submittal packages.
- Workflow templates and workflow steps.
- Sequential and parallel workflow routing.
- Dynamic approver due dates excluding weekends.
- Approver responses and final/official response.
- Return, close, revise, void/superseded status scaffolding.
- Linked drawings, RFIs, documents, spec sections, locations, companies, users.
- Attachments as metadata in the current static MVP.
- Audit log, notifications, reports, CSV export, and AI suggestion panel.

## Statuses

Submittal statuses: Draft, Required, Open, Submitted, Under Review, Returned, Approved, Approved as Noted, Revise and Resubmit, Rejected, Closed, Closed Revised, Void, Superseded.

Reviewer response statuses: Pending, Submitted, Approved, Approved as Noted, Revise and Resubmit, Rejected, No Exceptions Taken, Make Corrections Noted, Reviewed, Not Reviewed, Forwarded, Returned.

Package statuses: Draft, Open, Partially Submitted, Submitted, Under Review, Partially Returned, Returned, Closed, Void.

## Completion criteria

Complete when the MVP can create draft/required/submitted submittals, validate required fields, prevent duplicate numbers, assign ball in court, run sequential and parallel workflows, track approver due dates/overdue, collect responses, mark official response, return/close/revise, preserve history, link drawings/RFIs/documents, track packages/distribution, scaffold notifications, audit actions, export CSV, show dashboard metrics, search/filter/sort, seed data, pass tests, and include documentation.
