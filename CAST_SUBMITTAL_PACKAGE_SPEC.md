# CAST Submittal Package Spec

Packages group submittals by spec section, contractor, or procurement/closeout need. Package page/log shows package number, title, status, spec section, responsible contractor, manager, due date, item counts, submitted/open/returned/closed/overdue counts, dates submitted/returned/closed, and CSV export.

Actions scaffolded: create package, add/remove submittals, submit package, return package, close package, export package log, export package transmittal.
