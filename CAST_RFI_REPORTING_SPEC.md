# CAST RFI Reporting Spec

Reports:

1. Open RFI Log
2. Closed RFI Log
3. Overdue RFI Log
4. RFI Aging Report
5. Ball in Court Report
6. RFI by Responsible Contractor
7. RFI by Drawing
8. RFI by Spec Section
9. RFI with Cost Impact
10. RFI with Schedule Impact
11. RFI Average Response Time
12. RFI Completion Rate

Each report supports filters by project, status, assignee, company, ball in court, due date, overdue, drawing, spec section, location, cost impact, schedule impact, and priority. CSV export is required in MVP; PDF export is scaffolded.
