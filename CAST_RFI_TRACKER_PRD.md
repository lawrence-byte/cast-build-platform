# CAST RFI Tracker PRD

## Product

**Project:** CAST Project Controls Platform  
**Initial module:** RFI Tracking and Completion

## Goal

Give CAST a fast construction operations register for RFIs: create, route, assign, answer, mark official response, close, reopen, revise, link drawings/documents, export logs, and report on overdue/aging/ball-in-court conditions.

## MVP users

- Owner Admin
- CAST Admin
- Project Manager
- Project Engineer
- Architect
- Consultant
- General Contractor
- Subcontractor
- Read Only Viewer

## Core workflows

A. Create RFI as draft  
B. Submit RFI for review  
C. Open RFI  
D. Assign RFI to one or more assignees  
E. Track ball in court  
F. Add drawing references  
G. Add specification references  
H. Add location references  
I. Add attachments  
J. Add comments  
K. Add responses  
L. Mark one response as official  
M. Close RFI  
N. Reopen RFI  
O. Revise RFI  
P. Export RFI log  
Q. Search and filter RFIs  
R. Generate overdue RFI report  
S. Generate open RFI report  
T. Generate closed RFI report  
U. Generate RFI aging report  
V. Generate ball in court report

## MVP screens

1. RFI Dashboard with summary cards and reports.
2. RFI Log with search, filter, sort, and CSV export.
3. Create RFI form with draft/open validation.
4. RFI Detail panel with responses, comments, attachments, official response, revision history, audit log, and actions.
5. AI helper panel for suggestions only.
6. Drawing Log scaffold.
7. Document Register scaffold.

## Completion criteria

The MVP is complete when a user can create draft/open RFIs, validate required fields, prevent duplicate numbers, assign ball-in-court, respond, mark official response, close, reopen, revise, preserve revisions, link drawings/documents, track distribution list, scaffold notifications, create audit logs, export CSV, view metrics, search/filter/sort, load seed data, and pass tests.
