# CAST Document Tracker Spec

## Document fields

project_id, document_number, title, document_type, discipline, status, current_revision, file_url, permission_group, created_by, created_at, updated_at.

## Revision fields

document_id, revision_number, revision_date, revision_description, file_url, submitted_by, reviewed_by, review_status, review_due_date, review_completed_date.

## Statuses

Draft, Submitted, Under Review, Approved, Approved As Noted, Revise and Resubmit, Rejected, Superseded, Archived.

## Requirements

Single source of truth, version control, permission controlled access, revision history, workflow review status, attach documents to RFIs, export document register.
