# CAST Build Project Template

Every project should have:

- `data/projects/<slug>.json` for project metadata and module status
- `public/projects/<slug>.html` for the project workspace page
- `docs/projects/<slug>.md` for durable notes and decisions
- explicit Procore connection state
- read/write guardrails
- project-specific data boundary

Alüm is the first implementation. Overlook should be cloned from it after validation.
